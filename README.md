# PBPS-Staff-Payroll-System

A Python Tkinter application developed for Pentre Bach Pre School (PBPS).

## Features

* Secure login system
* Main menu navigation
* Add staff records
* Search staff records
* Save staff details to a text file
* Payroll calculator
* Tax, National Insurance and Pension calculations
* Data validation using message boxes
* Multiple GUI windows using Tkinter

## Technologies Used

* Python
* Tkinter

## Project Versions

### Version 1

* Login system
* Main menu

### Version 2

* Staff record management
* Save staff details to a text file

### Version 3

* Payroll calculator
* Tax, National Insurance and Pension calculations
* Data validation

### Version 4

* Search staff records by Staff ID
* Automatically retrieve stored staff information
* Display employee details in the system
